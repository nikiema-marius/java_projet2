*****Description*****:
Ce programme Java gère les informations des étudiants, incluant leurs notes, pour calculer et afficher diverses statistiques de la classe. Il demande à l'utilisateur le nombre d'étudiants à gérer, puis lit les informations de chaque étudiant (matricule, nom, prénom, moyenne). Il affiche ensuite la moyenne de la classe, les étudiants admis, recalés, et ceux nécessitant un rattrapage.


*****Comment tester le programme***** :
1---Ouvrez un terminal ou une invite de commande.

2---Naviguez vers le répertoire contenant les fichiers source Java 

4. Compilez les fichiers Java en utilisant la commande suivante :
  -- javac Etudiant.java 
  -- GestionDesEtudiants.java
  -- java GestionDesEtudiants
